# php 学習
## phpとは
 - web開発でよく使用されるスプリクト言語

 - htmlに埋め込むことができる  
 -<?php >が開始タグ,?>が閉じタグ
 -例
 ```
 <p>
  <?php echo "php"; ?>入門
  </p>
```
- 開始タグを省略できる <? でOK
終了タグを省略できるが、省略すると、以降全てがphpの扱いになる

-phpのコメントの書き方は//でOK
複数行になるときは/** で開始、*/で終了
-文字列は.を使えば連結できる
```

<? echo "php"."入門"; ?>
```

## 変数について
- 変数とは値を入れておく箱
- 変数=値で定義
- 変数の先頭には$マークを付ける
```
$tenp =1; //変数tenpに１を代入
echo $tenp; //１が出る
```
- 変数を上書きできる
- 変数で足し算・連結できる
```
$foo = 'php';
$fooo = '入門';
echo $foo.$fooo;
//php入門と出力
```
- 変数はくり返し使える
```
$program = "php";
echo $program."入門";
echo $program."初回";
```
